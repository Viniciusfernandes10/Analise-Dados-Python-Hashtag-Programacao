# 📊 Análise de Cancelamento de Clientes (Churn)

Este projeto foi desenvolvido durante o **Intensivão de Python da Hashtag Programação**, utilizando uma base de dados disponibilizada no **Kaggle**.

O objetivo do projeto é **analisar o comportamento de clientes e identificar os principais fatores que levam ao cancelamento de serviços (churn)**.

---

# 🧠 Contexto do Problema

Você foi contratado por uma empresa com **mais de 800 mil clientes** para realizar um **projeto de análise de dados**.

Recentemente, a empresa percebeu que **uma grande parte de sua base de clientes está inativa**, ou seja, **clientes que já cancelaram o serviço**.

Essa situação está impactando diretamente os resultados da empresa.

Diante disso, o objetivo do projeto é:

- Entender **os principais motivos dos cancelamentos**
- Identificar **padrões de comportamento dos clientes que cancelam**
- Descobrir **ações estratégicas que podem reduzir a taxa de cancelamento**

---

# 🎯 Objetivos da Análise

- 📉 Identificar os fatores que mais influenciam o **cancelamento de clientes**
- 📊 Analisar padrões e tendências nos dados
- 🧮 Utilizar **Python para exploração e tratamento dos dados**
- 💡 Gerar **insights que ajudem na tomada de decisão da empresa**

---

# 🛠️ Tecnologias Utilizadas

- **Python 3**
- **Pandas** → Manipulação e análise de dados  
- **Plotly / Matplotlib** → Visualização de dados  
- **Jupyter Notebook** → Ambiente de desenvolvimento  

---

# 📂 Etapas do Projeto

## 1️⃣ Importação e tratamento dos dados
- Carregamento da base de dados
- Limpeza e organização das informações
- Tratamento de valores ausentes

## 2️⃣ Análise Exploratória de Dados (EDA)
- Identificação de padrões nos cancelamentos
- Comparação entre clientes ativos e inativos
- Análise de variáveis relevantes

## 3️⃣ Visualização de dados
- Criação de gráficos para facilitar a interpretação
- Identificação visual dos fatores que impactam o churn

## 4️⃣ Geração de insights
- Identificação dos principais motivos de cancelamento
- Sugestões de possíveis ações para redução do churn

---

# 📈 Possíveis Insights

A análise permite identificar fatores como:

- Tempo de contrato
- Tipo de plano
- Frequência de uso do serviço
- Perfil dos clientes que mais cancelam

Essas informações ajudam a empresa a **desenvolver estratégias para retenção de clientes**.                                                                                                                           ---

# 🚀 Resultados Esperados

Com essa análise, a empresa poderá:

- Reduzir a taxa de cancelamento
- Melhorar a retenção de clientes
- Aumentar a receita
- Tomar decisões baseadas em dados
